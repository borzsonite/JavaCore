1. Отношение = таблица, атрибут = колонка, кортеж = строка, мощность = кол-во строк, размерность = кол-во колонок, домен = ячейка.
2. Имя таблицы должно начинаться с буквы
3. Ключевое поле id,  является ОБЯЗАТЕЛЬНЫМ ЭЛЕМЕНТОМ каждой реляционной таблицы.
4. Ключевое поле  должно состоять  из двух частей: начинаться с названия таблицы, которой поле принадлежит, затем через подчеркивание необходимо указать id, например book_id
5. Типы данных в SQL: int, decimal(вещественное число в скобка указывается количество сиволов слева и справа от запятой), date, varchar(строка не более 255 символов)
6. Описание ключевого поля INT PRIMARY KEY AUTO_INCREMENT
7. Алгоритм создание таблицы:

    - название таблицы
    - скобка

    - запятая
    - имя поля
    - ....
    - закрываающая скобка

14. Математические функции:
    - ceiling(x) возвращает наименьшее целое число, большее или равное x. Например, ceiling(4.2)=5; ceiling(-5.8)=-5)
    - floor(x) возвращает наибольшее целое число, меньшее или равное x. Например, 	floor(4.2)=4; floor(-5.8)=-6
    - round(x, k) 	округляет значение x до k знаков после запятой, если k не указано – x округляется до целого. Например, round(4.361)=4; round(5.86592,1)=5.9
    - power(x, y) возведение x в степень y, например, power(3,4)=81.0
    - sqrt(x) квадратный корень из х
    - degrees(x) конвертирует значение x из радиан в градусы
    - radians(x) 	конвертирует значение x из градусов в радианы
    - abs(x) модуль числа
    - pi() число пи
15. В SQL реализована возможность заносить в поле значение в зависимости от условия. Для этого используется функция IF: IF(логическое_выражение, выражение_1, выражение_2), например: SELECT title, amount, price, IF(amount<4, price*0.5, price*0.7) AS sale FROM book;
