1. Отношение = таблица, атрибут = колонка, кортеж = строка, мощность = кол-во строк, размерность = кол-во колонок, домен = ячейка.
2. Имя таблицы должно начинаться с буквы
3. Ключевое поле id,  является ОБЯЗАТЕЛЬНЫМ ЭЛЕМЕНТОМ каждой реляционной таблицы.
4. Ключевое поле  должно состоять  из двух частей: начинаться с названия таблицы, которой поле принадлежит, затем через подчеркивание необходимо указать id, например book_id
5. Типы данных в SQL: int, decimal(вещественное число в скобка указывается количество сиволов слева и справа от запятой), date, varchar(строка не более 255 символов)
6. Описание ключевого поля INT PRIMARY KEY AUTO_INCREMENT
7. Алгоритм создание таблицы:
    - ключевое слово CREATE TABLE (можно маленькими буквами)
    - название таблицы
    - скобка
    - имя поля (атрибута) + его описание, которое включает тип поля и другие необязательные характеристики
    - запятая
    - имя поля
    - ....
    - закрываающая скобка
8. Запрос можно писать на нескольких строках в конце точка с запятой, если запрос один - необязательо
9. Алгоритм вставки значений в таблицу:
    - ключевые слова INSERT INTO
    - имя таблиц
    - (
    - список полей в которые будут вноситься значения
    - )
    - ключевое слово VALUES
    - (
    - значения
    -)
10. Алгоритм выбора всех записей из таблицы: SELECT * FROM table_name
11. Алгоритм выбора нескольких записей из таблицы: SELECT имя_столбца1, имя_столбца2, итд. FROM table_name
12. Алгоритм выбора отдельных столбцов и присвоение им новых имен:
    - SELECT имя_столбца AS новое_имя, имя_столбца AS новое_имя, etc.. FROM table_name
13. Алгоритм  выборки данных с созданием вычисляемого столбца:
    - SELECT имя_столбца1, имя_столбца2, итд. имя_столбца1 * имя_столбца2 AS новое_имя FROM table_name
14. Математические функции:
    - ceiling(x) возвращает наименьшее целое число, большее или равное x. Например, ceiling(4.2)=5; ceiling(-5.8)=-5)
    - floor(x) возвращает наибольшее целое число, меньшее или равное x. Например, 	floor(4.2)=4; floor(-5.8)=-6
    - round(x, k) 	округляет значение x до k знаков после запятой, если k не указано – x округляется до целого. Например, round(4.361)=4; round(5.86592,1)=5.9
    - power(x, y) возведение x в степень y, например, power(3,4)=81.0
    - sqrt(x) квадратный корень из х
    - degrees(x) конвертирует значение x из радиан в градусы
    - radians(x) 	конвертирует значение x из градусов в радианы
    - abs(x) модуль числа
    - pi() число пи
15. В SQL реализована возможность заносить в поле значение в зависимости от условия. Для этого используется функция IF: IF(логическое_выражение, выражение_1, выражение_2), например: SELECT title, amount, price, IF(amount<4, price*0.5, price*0.7) AS sale FROM book;
16. С помощью ключевого слова WHERE в выборку можно включать строки отвечающиие определенному условию, например: SELECT title, author, price * amount AS total FROM book WHERE price * amount > 4000
17.  Логическое выражение после ключевого слова WHERE кроме операторов сравнения  и выражений может включать  логические операции (И «and», ИЛИ «or», НЕ «not») и круглые скобки, изменяющие приоритеты выполнения операций. Например: SELECT title, author, price FROM book WHERE (author = 'Булгаков М.А.' OR author = 'Есенин С.А.') AND price > 600;
18.Операции имеют следующий приоритет:
   1. круглые скобки
   2. умножение  (*),  деление (/)
   3. сложение  (+), вычитание (-)
   4. операторы сравнения (=, >, <, >=, <=, <>)
   5. NOT
   6. AND
   7. OR
19. Интервал можно задавать с помощью слов BETWEEN IN, например BETWEEN 5 AND 10 или IN(1,2,3,4). Пример запроса: SELECT title, author, price FROM book WHERE price BETWEEN 540.50 AND 800 AND amount IN(2,3,5,7)
20. Выборку можно делать с помощью ключевого слова LIKE и шаблонов. Шаблоны следующие: % - любая строка содержащая ноль и более символов, _ - любой одиночный символ.
